## Elastic

https://www.elastic.co/docs/get-started/howto-use-the-docs

### 
*Elastic* — платформа поиска, аналитика и ИИ.

Он предлагает 3 основных решения:
- *Elasticsearch* — система быстрого поиска, анализа и визуализации
- *Elastic Observability* — наблюдать и контролировать состояние и производительность системы
- *Elastic Security* — мониторинг данных на предмет аномальной активности

+ *Opensearch* — популярный fork amazon от Elasticsearch

*Elastic Stack* (ELK) — набор компонентов вокруг Elastic, которые позволяют безопасно извлекать данные, выполнять поиск, анализ и визуализацию

Популярные компоненты:
- *Kibana* — веб-интерфейс для визуализации данных из Elasticsearch и управления стеком Elastic
- *Beats* — лёгкая система сбора данных и отправка их в Elasticsearch

- *Elastic Agent* — более универсальная и гибкая система по сравнению с Beats
- *Fleet* — система централизованного управления Elastic Agents
- *Logstash* — система сбора данных со сложной обработкой (объединение, нормализация, фильтрация и т. д.) конвейером и отправкой в Elasticsearch
- *APM* — система мониторинга производительность, собирает метрики (время ответа, запроса данных, http запросы) и отправляет их в Elasticsearch
- *Elasticsearch clients* - клиенты для популярных языков программирования для управления запросами и ответами API Elasticsearch
